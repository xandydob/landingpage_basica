# 🌐 Minha Landing Page Semântica

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

Uma Landing Page moderna e totalmente responsiva desenvolvida com o objetivo de aplicar as melhores práticas de **HTML5 Semântico** e estilização avançada com **CSS3 Layouts** (Grid e Flexbox). 

O projeto conta com comentários didáticos no código HTML detalhando a função de cada tag, ideal para fins de estudo ou template base estruturado.

---

## 🚀 Funcionalidades e Destaques

- **HTML5 100% Semântico:** Uso correto de tags como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` e `<footer>`, garantindo uma excelente estrutura para SEO e acessibilidade.
- **Layout Responsivo:** Adaptável para dispositivos móveis, tablets e desktops utilizando *Media Queries*.
- **CSS Grid & Flexbox:** Uso de técnicas modernas para distribuição e alinhamento de componentes (como a seção de vantagens em formato de grid auto-ajustável).
- **Segurança na Navegação:** Links configurados com `target="_blank"` e `rel="noopener noreferrer"` para abertura segura de links externos.
- **Efeitos Visuais Fluídos:** Animações sutis de *hover* nos botões, links do menu e cartões de vantagens (`transform` e `transition`).
